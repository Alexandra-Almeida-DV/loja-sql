# Projeto SQL - Loja

Este projeto contém scripts SQL para criação e popular o banco de dados de uma loja.

## Arquivos

- `schema.sql`: Criação do banco de dados e tabelas
- `seed.sql`: Inserção de dados fictícios
- `views.sql`: Criação de views
- `procedures.sql`: Procedures e funções

## Banco de Dados

Compatível com SQL Server
